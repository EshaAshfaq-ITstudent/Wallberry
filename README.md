# 🍓 Wallberry - Aesthetic Wallpaper Hub

Wallberry is a modern, aesthetic Android application built with **Jetpack Compose** that allows users to personalize their devices with custom images and live wallpapers.

## ✨ Features
- 📸 **Camera Integration**: Snap a photo and instantly preview it as your wallpaper.
- 🖼️ **Gallery Picker**: Choose your favorite aesthetic shots from your local storage.
- 💖 **Favorites**: Save your top picks to a dedicated "Saved" collection.
- 🪄 **Live Wallpaper**: Support for dynamic, interactive live wallpapers.
- 🎨 **Material 3 Design**: A beautiful, warm-toned UI with smooth transitions and edge-to-edge support.

## 🛠️ Tech Stack
- **Language**: Kotlin
- **UI Framework**: Jetpack Compose (Material 3)
- **Image Loading**: Coil
- **Architecture**: Modern Android Architecture components

## 🚀 Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/Ayeshaismail5/Wallberry.git
   ```

2. Open the project in **Android Studio**.
3. Build and run on an Android device (API 24+).
